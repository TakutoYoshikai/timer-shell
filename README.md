# timer-shell
The timer to execute shell command.

### Requirements
* macOS or Ubuntu
* Node.js
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/timer-shell
```

**set timer**
```bash
timer-shell <HOUR> <MINUTE> <SHELL COMMAND>
# set timer at next 20:15
timer-shell 20 15 echo hello world
```
